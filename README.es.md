## Descripción general
Este proyecto es una **tarjeta de resumen de suscripción** para un servicio ficticio de música en streaming.  
Permite a los usuarios revisar su plan seleccionado, cambiarlo si lo desean, continuar al pago o cancelar el pedido.

La interfaz incluye:
- Una UI moderna y limpia  
- Diseño responsivo para distintos dispositivos  
- Estados de hover y focus accesibles  
- Tipografía y diseño consistentes con estándares profesionales  

## El desafío
Los usuarios deben poder:

- Revisar su resumen de pedido e identificar el plan de suscripción seleccionado.  
- Hacer clic en el botón **Proceed to Payment**.  
- Hacer clic en el botón **Cancel Order**.  
- Ver los estados hover en todos los elementos interactivos.  
- visualizar correctamente el diseño tanto en pantallas de escritorio como móviles.

## Mi proceso
1. Comencé estructurando el layout usando **contenedores semánticos en HTML**.  
2. Apliqué **Flexbox** y **CSS Grid** para alinear y centrar los componentes de manera eficiente.  
3. Utilicé **variables CSS** para mantener una gestión de colores limpia y reutilizable.  
4. Importé una tipografía desde Google Fonts para coincidir con el diseño del reto.  
5. Añadí estilos para los estados hover y focus, mejorando la accesibilidad y la retroalimentación visual.  
6. Implementé un **diseño completamente responsivo**, ajustando el fondo y los espacios para pantallas pequeñas.

## Tecnologías utilizadas
- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  

## Lo que aprendí
- Cómo organizar y estructurar CSS utilizando la **convención de nombres BEM** para mayor escalabilidad.  
- Cómo trabajar con **propiedades personalizadas de CSS (variables)** para un mantenimiento y gestión de temas más eficientes.  
- Cómo centrar contenido usando la propiedad **place-content de Grid** y el **espaciado gap de Flexbox**.  
- Cómo aplicar **imágenes de fondo responsivas** según el tamaño de la pantalla.  
- Cómo construir botones interactivos con efectos hover y contraste de color adecuado.
